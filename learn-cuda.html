learn CUDA

CUDA is hard to learn to be honest, and if you read https://docs.nvidia.com/cuda/cuda-c-programming-guide/, things can be complicated, which is true, CUDA is complicated.

That guide is good to be honest, but if you are totally new, nothing you can run there, for example from https://docs.nvidia.com/cuda/cuda-c-programming-guide/#kernels, 

```cuda
// Kernel definition
__global__ void VecAdd(float* A, float* B, float* C)
{
    int i = threadIdx.x;
    C[i] = A[i] + B[i];
}

int main()
{
    ...
    // Kernel invocation with N threads
    VecAdd<<<1, N>>>(A, B, C);
    ...
}
```

What is this `__global__`, `threadIdx` and `<<<>>>` operations right?

1. `__global__` reserved for kernel, https://docs.nvidia.com/cuda/cuda-c-programming-guide/index.html#global

2. `threadIdx` is programming representation for threads in CUDA, https://docs.nvidia.com/cuda/cuda-c-programming-guide/#thread-hierarchy

3. `<<<>>>` is to invoke kernel operation, https://docs.nvidia.com/cuda/cuda-c-programming-guide/index.html#kernels

You read it you will understand, this is thing is hard but that is the whole point of CUDA.
